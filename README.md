### 🔧Environment
  - Server : Ubuntu 18.04 LTS, Apache Tomcat 9.0
  - Framework : Spring 4.0, Java 14.0.2
  - Database : MySQL
  - Aws : m5.4xlarge > TCP > SSH > SCP > RDP
  - JDK-14.0.2
  - Nodejs
  - Python3

<br>
